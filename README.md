Ryzen-Monitor@psyredop is an extension for GNOME that displays processor (CPU) load, graphics core (GPU) load, and memory (RAM) usage in the top menu bar of the GNOME Shell.

Ryzen-Monitor@psyredop это расширение для GNOME с отображением загрузки процессора, загрузки графическго ядра и использования памяти в верхней строке меню GNOME Shell.
