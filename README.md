Ryzen-Monitor@psyredop is an extension for GNOME AMD Ryzen 7 7730U that displays processor (CPU) load, graphics core (GPU) load, and memory (RAM) usage in the top menu bar of the GNOME Shell. Expansion was done for yourself and your system.
